# Игра "Реверси" ("Отелло")
Версия 1.2

Автор: Мутыгуллин Руслан(sounditbox@gmail.com)

## Описание
Данное приложение является реализацией игры "Реверси"

## Требования
* Python версии не ниже 3.6
* TKinter
* Pickle


## Состав
* Графическая версия: `main.py`
* Консольная версия: `cmain.py`
* Тесты: `tests.py`
* ИИ: `ai.py`
* GUI: `gui.py`
* Cохранения: `pvp.pickle`, `pve.pickle`, `eve.pickle`
* Логика: `logic.py`


## Графическая версия
Запуск игры: `./main.py`
## Консольная версия
Запуск игры: `./cmain.py`

Ключами можно выбрать режим игры и уровень искусственного интеллекта 

## Особенности
* Присутствует возможность сохранения
* Три режима игры: PvE, PvP, EvE
* Три уровня искуственного интеллекта
